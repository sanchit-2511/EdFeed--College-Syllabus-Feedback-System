# EduFeed: College Syllabus Feedback System 🎓

EduFeed is a comprehensive web-based platform designed to bridge the gap between students, faculty, and administration regarding curriculum effectiveness. It allows for real-time feedback collection, syllabus visualization, and detailed administrative reporting.

## 🚀 Features

### For Students & Teachers
* **Syllabus Viewer:** Browse department-specific subjects and topics before providing feedback.
* **Interactive Feedback Form:** A step-by-step guided form with progress tracking.
* **Role-Based Access:** Unique interfaces for Students, Internal Teachers, and External Teachers.

### For Administrators
* **User & Department Management:** Full CRUD operations for managing college hierarchy.
* **Syllabus Management:** Dynamically update subjects and topics.
* **Feedback Analytics:** Visual reports (Line charts/Tables) showing average ratings and response trends.
* **Data Export:** Export feedback data to CSV or PDF for offline analysis.
